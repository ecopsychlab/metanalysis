# An S7 object to organised a meta-analysis

An S7 object to organised a meta-analysis

## Usage

``` r
meta_study(x)
```

## Arguments

- x:

  `Character scalar` pointing to the relevant folder.

## Examples

``` r

meta_study
#> <metanalysis::meta_study> class
#> @ parent     : <S7_object>
#> @ constructor: function(x) {...}
#> @ validator  : <NULL>
#> @ properties :
#>  $ folder_name: <character>
#>  $ study_names: <ANY>      
#>  $ study_data : <ANY>      
```
